# Toldos Los Pedroches - Sitio Web

Sitio web corporativo para Toldos Los Pedroches, empresa especializada en toldos, pérgolas y protección solar en Pozoblanco y el Valle de los Pedroches.

## Estructura del Proyecto

```
toldos_los_pedroches/
├── index.html          # Página de inicio
├── nosotros.html       # Página sobre nosotros
├── trabajos.html      # Galería de trabajos realizados
├── contacto.html      # Formulario de contacto
├── css/
│   └── style.css      # Estilos principales
├── js/
│   └── main.js        # JavaScript para interactividad
└── img/
    ├── logo.svg       # Logo de la empresa
    ├── favicon.svg    # Favicon
    ├── icono/
    │   ├── toldos.svg
    │   ├── pergolas.svg
    │   ├── cortinas.svg
    │   └── proteccion_solar.svg
    └── trabajos/      # Imágenes de trabajos realizados
```

## Características

- Diseño responsive y moderno
- Color corporativo azul (#1e3a8a)
- Menú de navegación con las secciones: Inicio, Nosotros, Trabajos, Contacto
- Logo formal que puede usarse como favicon
- Formulario de contacto funcional
- Diseño similar al sitio de referencia pero adaptado a toldos

## Uso

Simplemente abre `index.html` en tu navegador para ver el sitio web.

## Personalización

- **Colores**: Los colores corporativos están definidos en `css/style.css` en la sección `:root`
- **Contenido**: Edita los archivos HTML para modificar el contenido
- **Imágenes**: Reemplaza las imágenes placeholder en `img/trabajos/` con fotos reales de trabajos
